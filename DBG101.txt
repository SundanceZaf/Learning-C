Commands for GDB Debugger

1. gdb (filename)
   list - displays source code



To make break points
2. b
ex. b main - this will make a break at start of main 

3. f - this command shows you what line you are in the program

4. next - takes you to next line

5. print (can be used to print variables)

6. s - to go inside functions

7. delete 1 - this would delete breakpoint 1

8. c - continue (used when using break points to switch between current location to next break)

9. nexti - (for next instruction)
   disass
   print (variable)

10.info reg (to see the registers)
